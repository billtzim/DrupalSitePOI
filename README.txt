Το openthes_drupal_zip περιέχει τα αρχεία του drupal site. Μετά την αποσυμπίεση κάτω απο www server φάκελο πρέπει να γίνουν ρυθμίσεις σχετικά με τη βάση στο αρχείο sites/default/settings.php

To opentess_db.sql έχει το backup της βάσης, ένα απλό restore φτάνει. Τα CREATE DB και USE εντολές στην αρχή του sql script έχουν γίνει comment out αλλα υπάρχουν για όποιον τα χρειαστεί
